# 📊 my-stock-team

**종목명만 입력하면, AI 애널리스트 팀이 알아서 분석 리포트를 만들어 주는 프로젝트입니다.**

예) `"삼성전자 분석해줘"` 라고 말하면 → 재무·차트·뉴스·리스크를 분석해서 → 발표용 PPTX 리포트가 나옵니다.

> ⚠️ 학습/실습용 프로젝트입니다. **투자 권유가 아닙니다.**

---

## 🤔 어떻게 동작하나요?

종목명을 주면, 역할이 다른 4명의 AI 애널리스트가 협업합니다.

| 애널리스트 | 하는 일 | 데이터 |
|---|---|---|
| 🧾 **펀더멘털** | 재무·실적·공시 분석 | DART 전자공시 |
| 📈 **시장/기술** | 주가·추세·거래 동향 | 주가 데이터 |
| 📰 **뉴스/심리** | 뉴스·이슈·시장 분위기 | 웹 검색 |
| 🛡️ **리스크 매니저** | 위 3명 결과를 모아 위험 점검·종합 | 위 결과 종합 |

**진행 순서**
1. 펀더멘털·시장기술·뉴스 애널리스트 3명이 **동시에** 분석
2. 끝나면 리스크 매니저가 결과를 **모아서 종합**
3. 아래 **5부 구성** 리포트로 정리

> 표지 → 재무 → 차트 → 리스크 → 종합의견

## 📁 폴더 구조

```
my-stock-team/
├─ CLAUDE.md                  # 프로젝트 규칙 (팀 구성·작성 규칙)
├─ .claude/
│  ├─ agents/                 # AI 애널리스트 4명 정의
│  └─ skills/report-pptx/     # 리포트를 PPTX로 만드는 기능
├─ reports/                   # 분석 결과 (예: 삼성전자.md, .pptx)
└─ docs_cache/                # 데이터 캐시
```

---

## 🚀 시작하기

1. 이 저장소를 받습니다.
   ```bash
   git clone https://github.com/da02real/my-stock-team.git
   ```
2. `.env` 파일을 만들고 DART API 키를 넣습니다. (키는 [DART](https://opendart.fss.or.kr/)에서 무료 발급)
   ```
   DART_KEY=여기에_본인_키
   ```
3. AI 에이전트에게 분석을 요청합니다.
   ```
   삼성전자 분석 리포트 만들어줘
   ```

> 🔒 `.env`(API 키)는 `.gitignore`로 제외되어 GitHub에 올라가지 않습니다.

## 📌 분석 관점

- **가치투자(Value Investing)** 기준 — 기업의 내재가치와 펀더멘털 중심
- 모든 수치는 **출처와 기준일**을 함께 표기
- 매수/매도 단정 표현은 하지 않습니다 (판단은 독자의 몫)
